## 如何参与贡献

欢迎通过问题反馈、功能建议、文档改进或代码提交等方式参与贡献。请在上述仓库中提交 Issue 或 Pull Request。贡献指南、行为准则与参与流程详见仓库说明，期待你的参与！

也欢迎大家按 [路线图](/docs/roadmap) 参与贡献：标记为 **征集中** 的项（如新频道、模型提供商、Skills、MCP，或展示/交互优化等）都是很好的切入点。可先查看路线图中的当前重点与长期规划，再在 [GitHub Discussions](https://github.com/agentscope-ai/CoPaw/discussions) 参与讨论或认领任务。

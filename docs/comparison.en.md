# CoPaw vs OpenClaw: Feature Comparison

| Category            | Dimension                       | OpenClaw                                                                                                                                        | CoPaw                                                                                                                                                                                                                                                                                                |
| :------------------ | :------------------------------ | :---------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Tech Stack          | Primary Language                | TypeScript / Node.js                                                                                                                            | Python                                                                                                                                                                                                                                                                                               |
|                     | Agent Framework                 | Pi-agent SDK as the core agent engine                                                                                                           | • [AgentScope](https://github.com/agentscope-ai/agentscope)<br>• [AgentScope-Runtime](https://github.com/agentscope-ai/agentscope-runtime)                                                                                                                                                           |
|                     | Memory System                   | File-based memory (JSONL + MD)                                                                                                                  | File-based long-term memory system powered by [ReMe](https://github.com/agentscope-ai/ReMe)                                                                                                                                                                                                          |
| User Experience     | Installation                    | • npm install -g openclaw<br>• openclaw onboard CLI wizard                                                                                      | • .dmg, .exe installation<br>• One-line script installation<br>• pip install copaw<br>• Docker installation<br>• One-click cloud deployment                                                                                                                                                          |
|                     | Supported Platforms             | macOS / Linux / Windows (WSL2)                                                                                                                  | macOS / Linux / Windows (PowerShell/CMD)                                                                                                                                                                                                                                                             |
|                     | Self-hosting                    | Self-hosted via Docker                                                                                                                          | • Self-hosted via Docker (agentscope/copaw:latest, with Web Console)                                                                                                                                                                                                                                 |
|                     | Local Model Support             | Configure Ollama / llama.cpp endpoints via config file                                                                                          | On-demand integration via --extras during installation:<br>• --extras llamacpp (cross-platform)<br>• --extras mlx (Apple Silicon)<br>• --extras ollama (cross-platform, requires Ollama service)                                                                                                     |
|                     | Skills Support                  | • Supports local Skills<br>• Installation from ClawHub                                                                                          | • Supports local Skills<br>• Direct import from multiple community Skills Hubs<br>(Currently supported: https://skills.sh/..., https://clawhub.ai/..., https://skillsmp.com/..., https://github.com/...)                                                                                             |
|                     | Channel Integrations            | WhatsApp, Telegram, Slack, Discord, iMessage, Signal, Teams, Matrix, Zalo, etc.; extensible channel support                                     | DingTalk, Feishu, QQ, Discord, Telegram, iMessage, etc.; extensible channel support                                                                                                                                                                                                                  |
| Community Ecosystem | Open-source License             | MIT (fully open)                                                                                                                                | Apache 2.0 (business-friendly)                                                                                                                                                                                                                                                                       |
| Roadmap             | Memory System                   | Community-driven                                                                                                                                | • Multimodal memory fusion<br>• Experience distillation & Skill extraction<br>• Context-aware proactive delivery                                                                                                                                                                                     |
|                     | Multi-agent                     | Multiple isolated agents, plus multiple channel accounts in one running Gateway                                                                 | Native multi-agent isolation and collaboration through [AgentScope](https://github.com/agentscope-ai/agentscope)                                                                                                                                                                                     |
|                     | Self-healing                    | Community-driven                                                                                                                                | Daemon Agent for long-horizon tasks and health monitoring                                                                                                                                                                                                                                            |
|                     | Security                        | • Community-driven security hardening (WASM sandbox proposals, API key protection, etc.)<br>• ClawHub marketplace with full VirusTotal scanning | • Community-driven<br>• Tool-sandbox                                                                                                                                                                                                                                                                 |
|                     | Cloud-Native Architecture       | Community-driven                                                                                                                                | • Community-driven<br>• Continuously extend cloud capabilities via AgentScope Runtime, covering cloud compute, storage, and a broad cloud service ecosystem                                                                                                                                          |
|                     | Large-Small Model Collaboration | Not explicitly stated in official docs                                                                                                          | • Train and fine-tune locally deployable LLMs specifically for CoPaw's core native Skills and common tasks<br>• Lightweight local LLMs securely handle privacy-sensitive data; complex tasks (e.g. planning, coding) routed to powerful cloud LLMs — balancing security, performance, and capability |
|                     | Multimodal Interaction          | Community-driven                                                                                                                                | Voice and video interaction support                                                                                                                                                                                                                                                                  |
|                     | Community Ecosystem             | Community-driven; ClawHub continuously expanding                                                                                                | Continuously enrich the [AgentScope Skills](https://github.com/agentscope-ai/agentscope-skills) repository and improve the discoverability of high-quality Skills                                                                                                                                    |

# CoPaw 与 OpenClaw：功能对比

| 分类     | 维度         | OpenClaw                                                                                     | CoPaw                                                                                                                                                            |
| :------- | :----------- | :------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 技术栈   | 主要语言     | TypeScript / Node.js                                                                         | Python                                                                                                                                                           |
|          | 智能体框架   | 以 Pi-agent SDK 为核心智能体引擎                                                             | • [AgentScope](https://github.com/agentscope-ai/agentscope)<br>• [AgentScope-Runtime](https://github.com/agentscope-ai/agentscope-runtime)                       |
|          | 记忆系统     | 文件式记忆 (JSONL + MD)                                                                      | 基于文件的长期记忆系统，由 [ReMe](https://github.com/agentscope-ai/ReMe) 提供支持                                                                                |
| 使用体验 | 安装方式     | • npm install -g openclaw<br>• openclaw onboard CLI 向导                                     | • .dmg、.exe 安装包<br>• 一行脚本安装<br>• pip install copaw<br>• Docker 安装<br>• 一键云端部署                                                                  |
|          | 支持平台     | macOS / Linux / Windows (WSL2)                                                               | macOS / Linux / Windows（PowerShell/CMD）                                                                                                                        |
|          | 自托管       | 通过 Docker 自托管                                                                           | • 通过 Docker 自托管（agentscope/copaw:latest，含 Web 控制台）                                                                                                   |
|          | 本地模型支持 | 通过配置文件指定 Ollama / llama.cpp 端点                                                     | 安装时通过 --extras 按需集成：<br>• --extras llamacpp（跨平台）<br>• --extras mlx（Apple Silicon）<br>• --extras ollama（跨平台，需 Ollama 服务）                |
|          | 技能支持     | • 支持本地技能<br>• 从 ClawHub 安装                                                          | • 支持本地技能<br>• 从多个社区技能 Hub 直接导入<br>（当前支持：https://skills.sh/...、https://clawhub.ai/...、https://skillsmp.com/...、https://github.com/...） |
|          | 频道接入     | WhatsApp、Telegram、Slack、Discord、iMessage、Signal、Teams、Matrix、Zalo 等；可扩展频道支持 | 钉钉、飞书、QQ、Discord、Telegram、iMessage 等；可扩展频道支持                                                                                                   |
| 社区生态 | 开源协议     | MIT（完全开放）                                                                              | Apache 2.0（商业友好）                                                                                                                                           |
| 路线图   | 记忆系统     | 社区驱动                                                                                     | • 多模态记忆融合增强<br>• 经验蒸馏与技能提炼<br>• 上下文感知的主动推送                                                                                           |
|          | 多智能体     | 多智能体隔离，单 Gateway 内多频道账号                                                        | 通过 [AgentScope](https://github.com/agentscope-ai/agentscope) 原生多智能体隔离与协作                                                                            |
|          | 自愈         | 社区驱动                                                                                     | Daemon Agent 支持长周期任务与健康监控                                                                                                                            |
|          | 安全         | • 社区驱动的安全加固（WASM 沙箱提议、API Key 保护等）<br>• ClawHub 市场全量 VirusTotal 扫描  | • 社区驱动<br>• 工具沙箱                                                                                                                                         |
|          | 云原生架构   | 社区驱动                                                                                     | • 社区驱动<br>• 通过 AgentScope Runtime 持续扩展云能力，覆盖云算力、存储与云服务生态                                                                             |
|          | 大小模型协同 | 官方文档未明确说明                                                                           | • 针对 CoPaw 核心原生技能与常见任务训练与微调可本地部署的 LLM<br>• 轻量本地模型安全处理隐私敏感数据；规划、编码等复杂任务交由云端大模型 — 兼顾安全、性能与能力   |
|          | 多模态交互   | 社区驱动                                                                                     | 支持语音与视频交互                                                                                                                                               |
|          | 社区生态     | 社区驱动；ClawHub 持续扩展                                                                   | 持续丰富 [AgentScope Skills](https://github.com/agentscope-ai/agentscope-skills) 仓库，提升优质技能的发现与使用                                                  |

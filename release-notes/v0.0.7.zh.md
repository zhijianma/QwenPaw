## ✨ 新增功能

**安全**

- **Tool Guard 工具守卫**：在工具执行前进行安全检查，自动识别参数中的危险操作（如 Shell 命令里的 `rm`、`mv`）。发现风险后会拦截执行，等待用户通过 `/approve` 放行；拒绝列表中的工具则始终禁止调用。控制台新增**安全**设置页面，可管理检测规则和审批流程 ([#563](https://github.com/agentscope-ai/CoPaw/pull/563))

**频道与通信**

- **Mattermost 频道**：全新 Mattermost 集成，支持私聊和线程对话，可收发文本、图片、文件、视频和音频，并提供打字状态提示和灵活的访问控制 ([#1071](https://github.com/agentscope-ai/CoPaw/pull/1071))
- **Matrix 频道**：基于 matrix-nio 实现的 Matrix 协议接入，支持文本、图片、视频、音频和文件收发 ([#969](https://github.com/agentscope-ai/CoPaw/pull/969))
- **@提及过滤**：群聊中仅当机器人被 @提及（Telegram 还支持斜杠命令）时才会响应，可通过 `require_mention` 参数分频道配置，适用于 Discord、钉钉、飞书和 Telegram ([#1049](https://github.com/agentscope-ai/CoPaw/pull/1049))
- **Telegram Markdown 渲染**：将 LLM 输出的 Markdown 自动转为 Telegram HTML 格式，支持加粗、斜体、代码块、链接、引用、剧透等，发送失败时自动退回纯文本 ([#850](https://github.com/agentscope-ai/CoPaw/pull/850))
- **飞书表情回应**：消息处理成功后，自动在最后一条飞书回复上添加"DONE"表情 ([#1158](https://github.com/agentscope-ai/CoPaw/pull/1158))
- **飞书富文本媒体**：支持提取飞书 post 富文本消息中的媒体文件（视频、音频等），自动下载并作为附件传递 ([#1159](https://github.com/agentscope-ai/CoPaw/pull/1159))
- **QQ 图片消息**：智能体输出中可使用 `[Image: url]` 标签发送图片，私聊和群聊均可使用 ([#1212](https://github.com/agentscope-ai/CoPaw/pull/1212))

**模型与 AI 功能**

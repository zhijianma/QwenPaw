## ✨ Added

**Security**

- **Tool Guard**: Pre-execution security layer that scans tool parameters for risky patterns (e.g., `rm`, `mv` in shell commands). Risky calls are blocked until the user approves via `/approve`; denied tools are always blocked. New **Security** settings page for managing rules and approvals ([#563](https://github.com/agentscope-ai/CoPaw/pull/563))

**Channel & Communication**

- **Mattermost Channel**: Full Mattermost integration supporting DMs and threaded conversations with text, images, files, video, and audio, plus typing indicator and configurable access policies ([#1071](https://github.com/agentscope-ai/CoPaw/pull/1071))
- **Matrix Channel**: New Matrix protocol channel integration using matrix-nio, supporting text, images, video, audio, and file messages ([#969](https://github.com/agentscope-ai/CoPaw/pull/969))
- **Require Mention Filtering**: Group messages are ignored unless the bot is @mentioned (or a slash command is used on Telegram), configurable per-channel via `require_mention` for Discord, DingTalk, Feishu, and Telegram ([#1049](https://github.com/agentscope-ai/CoPaw/pull/1049))
- **Telegram Markdown Rendering**: Markdown-to-Telegram HTML conversion for LLM output including bold, italic, code blocks, links, blockquotes, spoilers, and more, with automatic plain text fallback on send failure ([#850](https://github.com/agentscope-ai/CoPaw/pull/850))
- **Feishu Emoji Reaction**: Automatically adds a "DONE" emoji reaction to the last Feishu reply when processing completes successfully ([#1158](https://github.com/agentscope-ai/CoPaw/pull/1158))
- **Feishu Rich Text Media**: Parsing of media files (videos, audio, and other files) in Feishu post-type rich text messages, with automatic download and attachment ([#1159](https://github.com/agentscope-ai/CoPaw/pull/1159))
- **QQ Image Messages**: Support for sending image messages via `[Image: url]` tags in agent output for both DM and group chats ([#1212](https://github.com/agentscope-ai/CoPaw/pull/1212))

**Model & AI Features**

- **Model Retry**: Automatic retries for LLM API calls on transient errors (rate limit, timeout, connection) with exponential backoff, configurable via `COPAW_LLM_MAX_RETRIES`, `COPAW_LLM_BACKOFF_BASE`, `COPAW_LLM_BACKOFF_CAP` ([#1150](https://github.com/agentscope-ai/CoPaw/pull/1150))
- **LM Studio Provider**: LM Studio added as a built-in model provider with configuration UI and documentation ([#1134](https://github.com/agentscope-ai/CoPaw/pull/1134))
- **Token Usage Tracking**: End-to-end token usage tracking across providers with a Token Usage settings page, API, and `get_token_usage` tool ([#775](https://github.com/agentscope-ai/CoPaw/pull/775), [#1349](https://github.com/agentscope-ai/CoPaw/pull/1349), [#1284](https://github.com/agentscope-ai/CoPaw/pull/1284))
- **Provider Advanced Configuration**: Added `generate_kwargs` editor for custom generation parameters (e.g., temperature, top_p, enable_thinking); providers without API keys (Ollama, LM Studio) are now correctly shown as configured ([#1170](https://github.com/agentscope-ai/CoPaw/pull/1170))

**Console & UI**

- **Workspace Drag-and-Drop**: Drag-and-drop reordering for enabled system prompt files in Agent Workspace, controlling prompt file precedence ([#1121](https://github.com/agentscope-ai/CoPaw/pull/1121))
- **Chat Model Selection**: Model selector dropdown on the Chat page for switching models during conversation ([#1140](https://github.com/agentscope-ai/CoPaw/pull/1140))
- **Agent Language Selector**: Change the agent's language directly from the console Agent Config page ([#1251](https://github.com/agentscope-ai/CoPaw/pull/1251))
- **Tool Batch Toggle**: "Enable All" and "Disable All" buttons on the Tools page for batch tool management ([#1112](https://github.com/agentscope-ai/CoPaw/pull/1112))
- **Chat URL Routing**: Direct URL access to specific chat sessions via `/chat/:chatId` with automatic URL synchronization ([#1221](https://github.com/agentscope-ai/CoPaw/pull/1221))
- **Context Management UI**: New Context Management section in Agent Config with adjustable compact ratio, reserve ratio, and tool result compaction settings, displaying derived token thresholds ([#1247](https://github.com/agentscope-ai/CoPaw/pull/1247))
- **Preserve Chat on Navigate**: Chat stays mounted when switching to other pages, preserving in-progress messages, cursor position, and running state ([#1340](https://github.com/agentscope-ai/CoPaw/pull/1340))

**Skills**

- **AI Skill Optimization**: "AI Optimize" button in the skill editor uses the active model to rewrite skill content with streaming output; supports English, Chinese, and Russian prompts ([#853](https://github.com/agentscope-ai/CoPaw/pull/853))
- **Skill Card Description**: Skill cards now display the `description` from SKILL.md frontmatter so you can see what each skill does without opening it ([#1235](https://github.com/agentscope-ai/CoPaw/pull/1235))

**Installation & Platform**

- **Auto PyPI Mirror**: Automatic PyPI mirror selection in the install script, detecting connectivity and falling back to Alibaba Cloud mirror for users in China ([#925](https://github.com/agentscope-ai/CoPaw/pull/925), [#1330](https://github.com/agentscope-ai/CoPaw/pull/1330))
- **Docker Secret Directory**: Added `/app/working.secret` to Docker layout for persistent provider settings and API keys ([#1050](https://github.com/agentscope-ai/CoPaw/pull/1050))

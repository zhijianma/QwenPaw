# 📋 Agent Kanban

一个基于 PawApp 框架开发的 AI 协作看板应用，支持多智能体任务管理与自动调度。

## ✨ 特性

- 🎯 **看板式任务管理** - 5列看板（待规划、等待调度、进行中、审核中、已完成）
- 🤖 **Agent 自动调度** - 智能分配任务给可用的 Agent
- 📊 **实时进度追踪** - SSE 实时流式输出任务执行过程
- ✅ **工具调用批准** - 支持需要人工确认的工具执行
- 🔄 **拖拽操作** - 通过拖拽在不同状态间移动任务
- 👥 **Agent 视图** - 切换到 Agent 维度查看任务分布

## 🚀 快速开始

### 安装

通过 QwenPaw 应用中心安装：

1. 打开 QwenPaw 应用中心
2. 搜索 "Agent Kanban"
3. 点击安装

### 使用

1. **创建任务** - 点击列标题的 `+` 按钮
2. **分配 Agent** - 在下拉菜单中选择 Agent
3. **运行任务** - 点击"运行"按钮启动执行
4. **查看结果** - 任务完成后点击"查看结果"查看详细输出
5. **拖拽移动** - 拖动卡片在不同状态间切换

## 📖 看板列说明

| 列名 | 说明 |
|------|------|
| 待规划 | 新创建的任务，等待编辑和规划 |
| 等待调度 | 已分配 Agent，等待执行 |
| 进行中 | Agent 正在执行中 |
| 审核中 | 任务完成，等待审核 |
| 已完成 | 任务已审核通过 |

## 🎨 视图切换

- **任务视图** - 按任务状态分组显示（默认）
- **Agent 视图** - 按 Agent 分组显示任务

## 🔧 技术栈

- **前端**: React + Ant Design
- **后端**: FastAPI + PawApp SDK
- **实时通信**: Server-Sent Events

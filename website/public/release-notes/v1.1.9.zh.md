## ✨ 新增功能

**桌面端与 IDE**

- **Tauri 2.x 桌面应用**：基于 Tauri 的全新原生桌面应用，支持 macOS 和 Windows ([#3813](https://github.com/agentscope-ai/QwenPaw/pull/3813))
- **Coding 模式**：全新 Web IDE，三栏布局——文件树、带行内 Diff 预览的多标签编辑器，以及集成分支管理、暂存和提交的 Git 面板 ([#4578](https://github.com/agentscope-ai/QwenPaw/pull/4578), [#4671](https://github.com/agentscope-ai/QwenPaw/pull/4671))

**技能与服务商**

- **技能市场**：新增技能市场页面，可从 Aliyun、ClawHub、ModelScope 等多个来源浏览、搜索和安装技能 ([#4518](https://github.com/agentscope-ai/QwenPaw/pull/4518), [#4623](https://github.com/agentscope-ai/QwenPaw/pull/4623))

- **无头浏览器验证提醒**：无头模式下遇到验证页面时，会提示用户切换到有界面模式 ([#4603](https://github.com/agentscope-ai/QwenPaw/pull/4603))

**Agent 系统**

- **后台任务超时**：`submit_to_agent` 新增 `task_timeout` 参数，CLI 同步新增 `--task-timeout` 选项 ([#4547](https://github.com/agentscope-ai/QwenPaw/pull/4547))
- **文件块上下文**：消息中的文件块现在和图片、音频、视频一样纳入上下文处理和 Token 计数 ([#4567](https://github.com/agentscope-ai/QwenPaw/pull/4567))
- **工具审批拒绝反馈**：用户拒绝工具调用后，Agent 会收到明确指令，不再重试或换用其他方案 ([#4569](https://github.com/agentscope-ai/QwenPaw/pull/4569))
- **定时任务隔离执行**：非共享定时任务在独立的干净上下文中运行，历史记录则持续累积在同一会话中 ([#4602](https://github.com/agentscope-ai/QwenPaw/pull/4602))

**聊天与控制台**

- **桌面端文件下载**：pywebview 桌面应用中，点击聊天文件卡片可正常下载 ([#4566](https://github.com/agentscope-ai/QwenPaw/pull/4566))
- **聊天输入草稿保存**：切换页面时自动保存输入框文本和光标位置，返回后恢复 ([#4598](https://github.com/agentscope-ai/QwenPaw/pull/4598))
- **聊天历史记忆摘要**：聊天历史现在包含记忆摘要 ([#4658](https://github.com/agentscope-ai/QwenPaw/pull/4658))

**频道**

- **统一访问控制**：支持按频道配置白名单/黑名单/待审批策略，并提供控制台界面统一管理各频道的用户权限 ([#4565](https://github.com/agentscope-ai/QwenPaw/pull/4565))
- **QQ 工具审批交互卡片**：QQ 频道中的工具审批支持键盘按钮卡片，可直接点击批准或拒绝 ([#4667](https://github.com/agentscope-ai/QwenPaw/pull/4667))
